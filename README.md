# نظام إدارة المساعدات (AMS) - Assistance Management System

نظام إدارة المساعدات هو تطبيق ويب متكامل ومتطور لإدارة المساعدات الخيرية والاجتماعية في مصر. يوفر النظام إدارة شاملة ومفصلة للمستفيدين والمساعدات والمؤسسات والمشاريع مع نظام صلاحيات متقدم وواجهة مستخدم عربية بالكامل.

## 🚀 المميزات الرئيسية

### 🔐 نظام المصادقة والصلاحيات
- **نظام أدوار متقدم**: مدير عام، مدير فرع، موظف، معتمد، مستفيد
- **حماية الصفحات**: كل صفحة محمية حسب دور المستخدم
- **إدارة المستخدمين**: إضافة وتعديل وحذف المستخدمين
- **تتبع جلسات العمل**: تسجيل آخر دخول للمستخدمين

### 📊 لوحة التحكم التفاعلية
- **إحصائيات شاملة**: عدد المستفيدين، المساعدات، المؤسسات، المشاريع
- **مؤشرات الأداء**: إجمالي المبالغ المدفوعة والمعلقة والمعتمدة
- **تحليلات مفصلة**: متوسط قيمة المساعدة، توزيع الجنس، إحصائيات الأسر

### 👥 إدارة المستفيدين المتقدمة
- **تسجيل شامل**: الاسم الرباعي، الرقم القومي، العنوان، الهاتف
- **البيانات الاجتماعية**: الجنس، الديانة، الحالة الاجتماعية، عدد أفراد الأسرة
- **البيانات المالية**: الدخل الشهري، الحالة الاقتصادية
- **البحث والفلترة**: البحث بالاسم أو الرقم القومي أو العنوان
- **التصدير**: تصدير بيانات المستفيدين بصيغة Excel

### 💰 إدارة المساعدات الشاملة
- **أنواع متعددة**: طبية، أيتام، أرامل، ذوي الاحتياجات، تعليمية، أسر السجناء، مالية
- **حالات متابعة**: معلق، قيد المراجعة، معتمد، مدفوع، مرفوض
- **طرق دفع متنوعة**: تحويل بنكي، نقدي، حساب داخلي، فيزا
- **ربط بالمستفيدين**: كل مساعدة مربوطة بمستفيد محدد
- **تتبع المبالغ**: إجمالي المساعدات حسب النوع والحالة

### 🏢 إدارة المؤسسات والشركاء
- **أنواع المؤسسات**: خيرية، طبية، اجتماعية، تعليمية، تنموية
- **بيانات الاتصال**: العنوان، الهاتف، البريد الإلكتروني، الشخص المسؤول
- **الحسابات البنكية**: رقم الحساب للتحويلات المالية
- **إدارة الشراكة**: تتبع العلاقات مع المؤسسات الشريكة

### 📋 إدارة المشاريع الخيرية
- **أنواع المشاريع**: اجتماعي، طبي، تعليمي، تنموي
- **حالات المشاريع**: نشط، قيد التنفيذ، مكتمل، معلق
- **إدارة الميزانيات**: تتبع التكاليف والميزانيات المخصصة
- **الجدولة الزمنية**: تواريخ البداية والنهاية للمشاريع
- **ربط بالمؤسسات**: كل مشروع مرتبط بمؤسسة محددة

### 📄 نظام طلبات المساعدة
- **طلبات المستفيدين**: للمستفيدين لتقديم طلبات مساعدة جديدة
- **طلباتي**: للمستفيدين لمتابعة طلباتهم
- **طلبات الإدارة**: للمديرين والموظفين لمراجعة واعتماد الطلبات
- **سير العمل**: نظام موافقات متدرج حسب الصلاحيات

### 📁 إدارة ملفات المساعدات
- **تنظيم الملفات**: تجميع المساعدات في ملفات منظمة
- **تتبع المبالغ**: إجمالي المبالغ وعدد المستفيدين لكل ملف
- **حالات الملفات**: مكتمل، قيد الإعداد، معلق
- **إدارة الوثائق**: ربط الوثائق والملفات بالمساعدات

### 🏢 إدارة الفروع
- **فروع متعددة**: إدارة فروع مختلفة للمؤسسة
- **مديري الفروع**: تعيين مدير لكل فرع
- **بيانات الفروع**: العنوان، الهاتف، الحالة
- **توزيع المستخدمين**: ربط المستخدمين بالفروع المناسبة

### 📊 التقارير والتصدير المتقدم
- **تقارير PDF**: تقارير مفصلة بصيغة PDF
- **تصدير Excel**: تصدير البيانات بصيغة Excel للتحليل
- **تقارير مخصصة**: إمكانية إنشاء تقارير حسب المعايير المطلوبة

### ⚙️ الإعدادات والتخصيص
- **إعدادات النظام**: تخصيص أنواع المساعدات والمشاريع
- **إعدادات المؤسسة**: بيانات المؤسسة الرئيسية
- **تخصيص التقارير**: إعداد قوالب التقارير
- **إدارة الصلاحيات**: تخصيص صلاحيات المستخدمين

## 🛠️ التقنيات المستخدمة

### Backend Framework
- **ASP.NET Core 8.0**: إطار عمل ويب متطور من Microsoft
- **Entity Framework Core**: ORM لإدارة قاعدة البيانات
- **SQLite**: قاعدة بيانات خفيفة ومحمولة

### Frontend
- **Razor Pages**: صفحات Razor للواجهة الأمامية
- **Bootstrap**: إطار عمل CSS للتصميم المتجاوب
- **jQuery**: مكتبة JavaScript للتفاعل

### Database
- **SQLite**: قاعدة بيانات محلية خفيفة
- **Entity Framework Migrations**: إدارة هيكل قاعدة البيانات

## 📦 التثبيت والتشغيل

### المتطلبات
- .NET 8.0 SDK
- Visual Studio 2022 أو Visual Studio Code

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd AssistanceManagementSystem
```

2. **استعادة الحزم**
```bash
dotnet restore
```

3. **تحديث قاعدة البيانات**
```bash
dotnet ef database update
```

4. **تشغيل المشروع**
```bash
dotnet run
```
